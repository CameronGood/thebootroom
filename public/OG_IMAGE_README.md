# Open Graph Image

This file documents the requirement for the Open Graph image used in social media sharing.

## Requirements

- **Dimensions**: 1200x630 pixels (1.91:1 aspect ratio)
- **Format**: JPG or PNG
- **File name**: `og-image.jpg` (or `og-image.png`)
- **Location**: `/public/og-image.jpg`

## Content Suggestions

The Open Graph image should include:
- TheBootRoom logo/branding
- Tagline: "Find Your Perfect Ski Boots"
- Visual elements related to skiing/ski boots
- Maintain brand colors (#F5E4D0, #040404, #2B2D30)

## Current Status

The OG image file (`og-image.jpg`) needs to be created. Until it's created, the metadata will reference it, but social media platforms may not display it correctly.

## Tools for Creation

You can create this image using:
- Design tools (Figma, Adobe Photoshop, Canva)
- Online OG image generators
- Next.js Image Generation API (for dynamic generation)

Once created, place the file at: `public/og-image.jpg`
